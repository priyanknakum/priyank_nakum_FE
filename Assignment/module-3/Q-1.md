# Q-1

## Are the HTML tags and elements the same thing?

### Ans.

- no,tags and elements are not same.
  <br>
- **tags:-** tags are used to construct web-pages and markups.
Ex: ```<title> </title> ```
  <br>
- **Elements:-** Elements are made of opening tag, content, closing tag.
Ex: ```<b> this is bold </b>```
